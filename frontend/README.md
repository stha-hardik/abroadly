# Abroadly Frontend

Next.js (App Router) + TypeScript. Intentionally minimal — three pages, no styling library.

## Run

```
npm install
cp .env.local.example .env.local
npm run dev
```

## Pages

- `/` — entry
- `/onboarding` — student profile form
- `/chat` — chat + upload widget

API base is read from `NEXT_PUBLIC_API_BASE`.
