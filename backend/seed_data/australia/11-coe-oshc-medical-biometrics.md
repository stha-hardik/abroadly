# CoE, OSHC, Medicals, Biometrics, and Visa Lodgement

After admission, the Australia process moves from offer to payment, CoE, visa lodgement, health checks, biometrics if requested, and decision waiting. The local training cycle showed a practical sequence: offer, financial documentation, GS/provider approval, payment approval, NOC/NOL, institutional payment, CoE, visa lodgement, decision, and pre-departure.

## Confirmation of Enrolment

The CoE proves the student is enrolled in a CRICOS-registered course. It is usually issued after:

- Offer conditions are met.
- Offer acceptance is signed.
- Deposit or first instalment is paid according to the provider's official instructions.
- OSHC is arranged or provider-recorded.
- Provider is satisfied with GS/financial checks if it conducts them before CoE.

For packaged courses, there may be multiple CoEs. Include every CoE code in the visa application.

## Overseas Student Health Cover

Home Affairs requires OSHC for the student and included family members unless an exemption applies. OSHC should cover the full intended stay. If the student is outside Australia, the OSHC start date should be the date they arrive in Australia, not just the course start date. If the provider arranges OSHC, the student still needs insurer name and dates. If the student arranges it, they need the policy number for the visa application.

## Medicals and HAP ID

Students may need health examinations. Home Affairs can generate a HAP ID through ImmiAccount or related health process. Use only approved panel physicians listed by Home Affairs. Do not direct students to unofficial medical centres.

## Biometrics

Biometrics may be requested. If requested, the student follows the instructions in ImmiAccount and attends the nominated biometrics collection centre. Keep appointment letters and receipts.

## Visa Lodgement Package

Typical visa package includes:

- Online application in ImmiAccount.
- Passport.
- CoE or CoEs.
- GS answers and supporting documents.
- Financial capacity documents if required or prudent.
- OSHC details.
- English test evidence if required.
- Academic documents.
- Work/gap documents if relevant.
- Marriage/dependent documents if relevant.
- Health/medical and biometrics steps as instructed.
- Form 956A or representative forms only if the student is using a registered migration agent or authorized recipient. A self-lodging student does not need to appoint a representative.
